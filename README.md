# Atmoschem R-Universe

This repository serves as the official registry for the **Atmoschem R-Universe**. It automatically monitors, builds, and serves R packages developed by the [@atmoschem](https://github.com/atmoschem) organization.

By using this universe, you can install packages as **pre-compiled binaries** for Windows and macOS, avoiding the need for local compilation tools (like Rtools or Xcode).

## 🚀 Installation

To use the packages in this universe, you can add it to your R `repos` options.
